读书笔记

TotalCommand

文件操作

F3 预览文件
F4 用外部指定程序打开文件
F5 复制
F6 移动
F7 新建文件夹

批量修改文件名

快速跳转文件夹

alt + ↓ 弹出历史记录

ctrl + d 文件夹直达，以后调用方便，可以先打开要收藏的文件夹，然后直接ctrl+d即可

ctrl + q 进入预览模式，可以在右侧框中预览，再按一下退出

ctrl + b 这时所有的文件都不分层级的展示了出来

ctrl + z 给文件添加备注

alt + f1 磁盘直达（左窗口）， f2 右窗口
切换后可能在根目录，也可能在某一个访问过的目录，这取决于TC设置。建议原则：设为总到根目录。设置方法：菜单:配置→选项→操作→主程序→选中“更改驱动器时转到根文件夹”。

ctrl + s 快速搜索，输入字母以后，直接过滤显示文件夹
配置→选项→操作→快速搜索→字母（显示快速搜索框）

ctrl + → 左边的文件夹在右边打开

ctrl + t 打开新的标签页

ctrl + w 关闭标签页

ctrl + f1 切换文件列表现实方式

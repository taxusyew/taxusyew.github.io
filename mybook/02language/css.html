BFC (Box Formatting Context) 环境

三个特性
① 阻止垂直外边距（margin-top、margin-bottom）折叠
② 不会重叠浮动元素
③ 可以包含浮动

清除浮动
① 为父元素添加overflow:hidden;
② 同时浮动父元素
③ 在父元素添加class=”clearfix”类 

.clearfix:after{
    content: “.”;
    display: block;
    height: 0;
    visibility: hidden;
    clear:both;
}

使用BFC使用float的时候会使父容器长度缩短，div应用float后会根据内容来改变长度 还有个重要缺陷——父容器float解决了其塌陷问题，那么父容器的父容器怎么办？

IE6、7不支持伪元素 在IE+、现代浏览器上使用伪元素 在IE6、7使用hasLayout

.floatfix{
    *zoom:1;
}

reset css

为屏蔽各个浏览器默认的属性，使得自己的css在各平台下得到相同的效果

例如ul默认带有缩进的样式，在IE下，它的缩进是通过margin实现的，而Firefox下，它的缩进是由padding实现的

技巧

  1. 子星选择符 > *，选择所有的子元素，比后代选择器效率还高
  2. 如果按钮是背景图片的话，hover效果不是替换图片，可以添加白色背景，修改透明度 color: #fff; -moz-opacity: 0.9; opacity: 0.90; filter: alpha(opacity=90);
  3. a链接hover时，背景颜色改变时延时效果，-webkit-transition: color 300ms;
  4. 网站的shortcut，可以用png，也不一定非得是ico，
  5. hover的时候，背景突出放大，-webkit-transform: scale(1.05);
  6. css中的%是相对于父元素来说的
  7. 绝对定位元素的定位相对于最近已定位（有position属性）的祖先元素 所以如果想对一个元素使用局部的绝对定位，就对它的父元素使用position:relative;，这样不是让父元素移动，而是让子元素有个可以依靠的锚
  8. 任意浏览器的默认字体高都是16px，为了简化font-size的换算，需要在css中的body选择器中声明 Font-size=62.5%，这就使em值变为 16px*62.5%=10px http://www.cnblogs.com/showker/archive/2010/05/24/1742821.html
  9. 多行文字居中，table，黑岩的book_detail.html 中间的票数推荐，table天然就是垂直居中的
  10. 解决ios上面border-bottom底部两个像素，使用border-image

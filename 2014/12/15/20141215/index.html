初次安装

进入hexo项目的文件夹

执行

git clone <repository> themes/<theme-name>

然后修改 hexo/_config.yml

P82 theme: alberta // 修改为

然后重新生成

hexo generate
hexo server

打开localhost:4000看看就行

更新hexo

最近因为重新安装系统，所以重新安装了一遍node，但是进入hexo文件夹后，执行hexo generate就会报错，所以重新安装了一遍hexo

重新安装一遍hexo以后，会造成一个问题，hexo的配置文件_config.yml被修改了，把之前保存好的给复原回来
主要就是deploy项的参数，type由github换为git，repository换为repo即可

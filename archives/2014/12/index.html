RSA的使用

12月 29, 2014

RSA的使用
虽然原理比较复杂，但是实际用起来还是挺简单，今天就先从最简单、最实用的地方入手
具体原理，等待之后补充

  1. 选择两个不相等的质数p和q
  2. 计算 n = p x q
  3. 计算 s = (p-1) x (q-1)
  4. 从区间 (1, s) 中选择一个与 s 互质的数 e
  5. 通过运算 ed ≡ 1 (mod s) 求出数字 d
    即求解 ex + sy = 1 的解，得到的整数解 x ，即为 d

至此，所有的密钥计算都已经完成，可以得到两对密钥
公钥 (n, e)
私钥 (n, d)
自己留下私钥，把公钥发布出去，谁想给我发信息又不想让别人知道，就用(n, e)加密了发给我
别人用我的公钥加密信息m过程: m^e mod n -> c
我得到信息c以后的解密过程: c^d mod n -> m

2014读书清单

12月 16, 2014

今年快到年底了，虽然买了一大堆的书，可是真正看完的倒是没多少
所以总结一下都读了什么书，看看到底读书进度有多少

专业相关：

####《精通CSS：高级Web标准解决方案（第二版）》
有关浮动、定位的地方讲的非常透彻，最后几章的bug排查、hack也很有用

####《JavaScript核心概念及实践》
有很多不错的小实例代码，有关概念可以一目了然，最开始想买这本书是因为它专门有一部分讲JS引擎，但其实作用也不是很大

####《深入浅出Nodejs》

####《Vim实用教程》
很多的vim操作精华，本来以为自己的vim使用已经很熟练了，可是看看这本书讲的技巧，才发现原来操作应该是这样！

####《智能Web算法》
突然想学习机器学习方面的东西，就买来看看，有的收获也是表面的理论，代码是java的，所以看起来不费力气，但是没能实战一下

####《linkers and loaders》
很古老很古老的一本书，讲的编译器、链接器，还有老的程序格式究竟是怎么来的，有醍醐灌顶的感觉，但是又有相当大篇幅不是讲x86平台，有很多unix，spark平台的东西，拓展着看看也不错

《炒股的智慧》

历史：

hexo安装新主题

12月 15, 2014

初次安装

进入hexo项目的文件夹

执行

git clone <repository> themes/<theme-name>

然后修改 hexo/_config.yml

P82 theme: alberta // 修改为

然后重新生成

hexo generate
hexo server

打开localhost:4000看看就行

更新hexo

最近因为重新安装系统，所以重新安装了一遍node，但是进入hexo文件夹后，执行hexo generate就会报错，所以重新安装了一遍hexo

重新安装一遍hexo以后，会造成一个问题，hexo的配置文件_config.yml被修改了，把之前保存好的给复原回来
主要就是deploy项的参数，type由github换为git，repository换为repo即可

什么是简单线性回归

给定一个数据集合，取出其中一列值，作为解释变量 X，顾名思义，我们将用它来解释我们希望预估的变量 Y. 选取完解释变量后，我们使用选择学习算法，来通过 X 推测 Y，最终得到一个回归直线。 我们通常选择的学习算法是“最小二乘法”。 有关最小二乘法的详细解释可以查看：最小二乘法详细解释，在此不再赘述。

有关最小二乘法的误区

在没有真正的亲自手写一个回归算法的时候，我一直以为算法的运行流程如下：

  1. 设置回归的目标函数
  2. 每次使用一条数据，通过这个目标函数计算，优化一下参数
  3. 下次迭代使用新数据，新参数
  4. 最终在数据迭代完成以后，参数优化完成

后来在自己亲手实现算法的时候，发现并不是这样一个数据一个数据的迭代优化参数，而是拿着所有数据，整体的计算参数。 真实流程如下：

  1. 根据最小二乘法（OLS）计算出来回归直线的两个参数计算公式:
    \[ a = (\overline{X}\overline{Y} - \overline{XY}) \over (\overline{X})^2 - \overline{X^2} \]

\[ b = \overline{Y} - a \overline{X} \] 2. 可以看到为了得到这两个参数值，我们需要计算 4 个不同的均值： \[ \overline{X} \] \[ \overline{Y} \] \[ (\overline{X})^2 \] \[ \overline{X^2} \]

  1. 剩下的就是一次性使用全部数据，计算上面的 4 个值，然后带入第一步的公式，即可得到归回方程的系数

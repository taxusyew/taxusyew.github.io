像素单位总结

Screen size spx
Screen density
Resolution ppx

计算一个手机的 ppi，传入参数： x -> 屏幕物理像素宽， y-> 屏幕物理像素高， inch -> 屏幕物理对角线长度（英寸）

软件分辨率 = 硬件分辨率 * 分辨率比率

ppi = sqrt( 软件分辨率宽^2 + 软件分辨率高^2 ) / 屏幕对角线长度

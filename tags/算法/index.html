投色子正好到2015的概率

4月 03, 2015

今天有个朋友问我了一道算法题，刚开始以为挺简单，手动计算即可，可是后来发现还是挺复杂的，现在记录一下。

小明在玩一个扔骰子游戏，每次扔出等概率得到1到6的点数，现在他开始扔骰子，并把每次得到的点数累加(从0开始)，如果某一次他扔完之后的累加和恰好为2015则为成功，越过2015为失败(即某次从小于2015的某个数加完后变为大于2015的某个数)，小明成功的概率约为？

稍微分析以后就可以得到要到2015，有六种情况：2014+1，2013+2，2012+3，2011+4，2010+5，2009+6

01背包

3月 02, 2015

摘自维基的定义：

The knapsack problem or rucksack problem is a problem in combinatorial optimization: Given a set of items, each with a mass and a value, determine the number of each item to include in a collection so that the total weight is less than or equal to a given limit and the total value is as large as possible.

背包问题是组合优化的一种：给定一个集合，每个元素都有一定质量和价值，确定一个集合中元素的数量，使其总重量小于或等于给定的限制而总价值尽可能大。

RSA的使用

12月 29, 2014

RSA的使用
虽然原理比较复杂，但是实际用起来还是挺简单，今天就先从最简单、最实用的地方入手
具体原理，等待之后补充

  1. 选择两个不相等的质数p和q
  2. 计算 n = p x q
  3. 计算 s = (p-1) x (q-1)
  4. 从区间 (1, s) 中选择一个与 s 互质的数 e
  5. 通过运算 ed ≡ 1 (mod s) 求出数字 d
    即求解 ex + sy = 1 的解，得到的整数解 x ，即为 d
